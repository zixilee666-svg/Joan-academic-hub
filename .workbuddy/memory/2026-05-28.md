## 修复 KnowledgePage 未定义错误 (00:30)

### 问题
部署后访问 `/dashboard/knowledge` 报错：`Uncaught ReferenceError: KnowledgePage is not defined`

1. `src/App.tsx` 中使用了 `<KnowledgePage />` 但未导入该组件
2. `src/pages/KnowledgeGraphPage.tsx` 导入了 `getLocalKG` 但 `src/lib/knowledgeGraph.ts` 未 export 该函数

### 修复 (commit c76f00e)
- `src/App.tsx`：添加 `import KnowledgeGraphPage from '@/pages/KnowledgeGraphPage'`，并将两处 `KnowledgePage` 改为 `KnowledgeGraphPage`
- `src/lib/knowledgeGraph.ts`：将 `function getLocalKG()` 改为 `export function getLocalKG()`
