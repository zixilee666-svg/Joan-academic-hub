# 2026-05-28 工作日志

## 知识图谱边标签修复

**问题**: 知识图谱可视化中所有边的关系标签显示为 "undefined"。

**根因**: 数据层 (`data/knowledgeGraph.ts`) 使用字段名 `type` 存储关系类型（如 `{ source, target, type: 'is_a' }`），但可视化组件 `KnowledgeGraphCanvas.tsx` 的 props 定义和映射逻辑读取的是字段名 `relation`（`l.relation`）。字段名不匹配导致 `relation` 始终为 `undefined`。

**修复内容**:
1. `KnowledgeGraphCanvas.tsx` props 的 `links` 类型同时支持 `relation?: string` 和 `type?: string`
2. links useMemo 映射逻辑改为 `const relType = l.relation || l.type || ''`，优先取 `relation`，回退到 `type`
3. `RELATION_LABELS` 补充缺失的 `uses: '使用'` 映射

**涉及文件**:
- `src/components/knowledge/KnowledgeGraphCanvas.tsx`

**验证**: TypeScript `--noEmit` 零错误通过。
**GitHub**: commit `7636d69`，已推送到 `origin/main`。
