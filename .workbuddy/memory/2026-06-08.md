## TDZ 错误修复
- 错误：`Uncaught ReferenceError: Cannot access 'ye' before initialization`
- 根因：`selectAllFiltered` useCallback（第250行）引用了 `filtered`（第305行 useMemo），JS const TDZ 导致运行时崩溃
- 修复：将 `selectAllFiltered` 移到 `filtered` useMemo 之后定义

## 性能优化（Dashboard 加载慢）
- 根因分析：`/api/papers?pageSize=200` 耗时 10.6s，原因是 Edge Function `handleGetPapers` 对 159 篇论文逐条顺序 KV 读取 (for...of await kvGetJson)
- 修复1：`handleGetPapers` / `handleGetReadingStats` / papers export 中全部改为 `Promise.all(paperIds.map(...))` 并行读取
- 修复2：DashboardPage `loadData` 改为 `Promise.all([stats, papers, projects])` 并行发起，不再顺序等待
- 修复3：DashboardPage 增量渲染：不等待 stats 加载完成就可以显示页面，stats 卡片区域显示骨架屏
- 提交：`c058d35`
